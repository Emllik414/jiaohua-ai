# 饺滑浏览器插件安装说明

该插件需要与“饺滑-AI划词助手”桌面端同时运行，用于提高网页、字幕和复杂页面中的取词准确度。

## Edge

1. 在地址栏输入 `edge://extensions/`。
2. 开启“开发人员模式”。
3. 点击“加载解压缩的扩展”。
4. 选择解压后的插件文件夹。该文件夹中应直接包含 `manifest.json`。
5. 刷新普通网页后重新划词。

## Chrome

1. 在地址栏输入 `chrome://extensions/`。
2. 开启“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择解压后的插件文件夹。该文件夹中应直接包含 `manifest.json`。
5. 刷新普通网页后重新划词。

## 注意事项

- 桌面端必须保持运行。
- `edge://`、`chrome://`、浏览器扩展商店等受保护页面不能运行普通扩展。
- 免安装版首次启动时会自动将插件复制到稳定目录，并弹出安装引导。
